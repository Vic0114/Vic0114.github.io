六、客户地域分布

地域洞察

· 国内核心：广东（31.1%）+ 浙江（14.1%）+ 江苏（11.1%） = 三大服装产业带，占国内 72%

· 海外核心：南亚（孟加拉14条+巴基斯坦13条） = 27条，占比达海外总量的 58.7%，是服装代工转移的核心目的地

· 南非3条、印度2条、马来西亚2条，分散在多个国家，说明 TikTok 全球投流效果正在显现

七、询盘增长趋势

趋势判断：从 2025/8 开始进入稳定增长通道，2026/4 起加速放量（23条→17条→36条）。2026年6月单月36条，是去年同期的12倍，说明多渠道投流策略正在生效，但需同步加强精准度和转化能力。

九、运营改善建议

🎯 短期可执行：
1. 表格规范："行业"列填写率仅26.7%（36/135），建议设为必填下拉选项，分类后用数据指导投放
2. 渠道优化：百度推广关键词加否定词，过滤"汽车卡扣""糖果批发"等不相关流量
3. 跟进SOP：加微后 24h/3天/7天 三阶段自动提醒话术，降低沉默率（当前12%）
4. 官网强化：官网流量仅3.7%但成交了最高客单价（¥32,791），建议SEO投入和官网优化
📈 中长期战略：
1. 南亚特攻：孟加拉+巴基斯坦=58%海外客户，建议针对这两个市场做专项产品推荐（工字扣/撞钉是牛仔裤生产刚需）
2. 视频号+TikTok双引擎：合计23.6%且增长快，继续扩大投流，尤其 TikTok 表单和私信模式
3. 品类聚焦：四合扣/啪钮是最大需求品类（17%），且与箱包行业100%匹配，可作为对外展示的核心品类
4. 建立客户分级：将客户分为A类（服装品牌/有LOGO定制需求）、B类（贸易商）、C类（个人散单），差异化跟进策略
